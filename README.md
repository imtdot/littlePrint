# littlePrint 🖨️

A tiny, colorful, and convenient print utility for Python.

## Why littlePrint?

Tired of typing `print()` all the time? `littlePrint` gives you:

- **`pr()`** - Shorter than `print()`
- **`py()`** - Quick Python version info
- **Colorful output** - Make your terminal output beautiful
- **Convenient functions** - `info()`, `success()`, `warning()`, `error()`

## Installation

```bash
pip install littlePrint
